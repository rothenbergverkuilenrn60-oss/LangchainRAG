# LangchainRAG
Based on the RAG system in the upper half of the Merck manual.


Based on the RAG system in the first half of the Merck Manual of Clinical Practice, this system includes a rule engine and entity recognition, recursion, semantic segmentation, parent-child, summary, and metadata indexing, self-rag, query rewriting and expansion decomposition, Hyde, RRF, and Reranker, context processing compression and reordering, and multi-user support.
